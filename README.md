# 🚀 INSTRUCCIONES DE INSTALACIÓN (EL MÉTODO DE LA RECETA)

Este programa está construido en **código puro de Python** para garantizar que sea ligero, rápido y libre de los falsos positivos que suelen generar los antivirus con los archivos ejecutables (.exe) en entornos de oficina.

Para ponerlo a correr en tu máquina, solo debes seguir estos pasos:

### 📋 Requisitos Previos
* Tener Python 3 instalado en tu sistema.
* Tener acceso a la terminal de comandos (CMD o PowerShell).

### 🛠️ Pasos para "Cocinar" el Programa

1. **Descarga los archivos:** Asegúrate de bajar el script de Python (.py) de este repositorio.
2. **Abre tu terminal:** Navega desde la consola (CMD) hasta la carpeta donde guardaste el archivo.
3. **Ejecuta el programa:** Arranca el script con el comando:
   python Recuperador_Atomico.py

¡Y listo! El código correrá limpio, puro y sin restricciones de administrador.
